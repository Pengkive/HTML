list - 목록

목록의 종류

  • <ul> - 순서 없는 목록 (unordered list)
  • <ol> - 순서 있는 목록 (ordered list)
  • <dl> - 정의 목록 (description list)

1. 순서 없는 목록 - ul

web의 기본 기술 3가지

  • html - 웹문서의 구조를 담당
  • css - 웹문서의 디자인을 담당
  • javascript - 웹문서의 액션을 담당
  • red
  • green
  • blue

(문제) 추천하는 커피 메뉴 5가지를 순서없는 목록으로 만들자

  • 바닐라 샷
  • 카페모카
  • 카페라떼
  • 아인슈페너

  1. html - 웹문서의 구조를 담당
  2. css - 웹문서의 디자인을 담당
  3. javascript - 웹문서의 액션을 담당
(문제) 추천하는 개봉영화 5가지를 순서있는 목록으로 만들자

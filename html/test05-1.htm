html5에서 추가된 폼요소들

input type="time" 시간(시,분)을 지정

input type="date" 날짜를 지정

input type="week" 주 단위 날짜를 지정

input type="month" 월 단위 날짜를 지정

input type="datetime-local" 연도, 월, 일, 시, 분을 지정

input type="tel" 전화번호, 모바일에서 숫자 키패드 띄우기

모바일 브라우저에서 가상키패드가 숫자 형식으로 바뀔수 있고 전화걸기 가능

input type="email"을 지정

모바일 브라우저에서 가상키패드가 영문포함하는 형식으로 바뀔수 있고 브라우저에서 이메일 형식이 맞는지 체크할 수 있다

input type="url"을 연락처(웹사이트주소)를 지정

모바일 브라우저에서 가상키패드가 영문(http://) 포함하는 형식으로 바뀔수 있고 브라우저에서 알맞은 형식인지(프로토콜 여부)를 체크함.

input type="number" 숫자 형식을 지정

숫자값을 포함하는 형식, 증감버튼이 생성 최소값, 최대값을 속성으로 지정할 수 있고 증감 간격도 속성으로 지정가능


min 0, max 20, step 2 지정

input type="range" 범위내의 값을 지정

스크롤로 범위내 값 지정

html5에서 추가된 속성들

autofocus - 자동포커싱(입력대기상태)된 요소로 지정

autocomplete = "on/off" - 자동완성 기능을 켜거나 끔


placeholder - 입력예시를 보여주거나 설명문을 넣어줌

readonly - 읽기만 가능한 요소로 지정

disabled - 사용불가한 요소로 지정

required - 필수입력 요소로 지정

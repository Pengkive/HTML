input type="submit/reset/button" 전송/초기화/일반버튼

input type="submit/reset"은 폼의 정보를 전송하거나 초기화하는 역할의 버튼, input type="button" 폼과는 무관한 일반 버튼이 생성됨
버튼의 글자는 value 속성으로 지정.

button..../button 전송/초기화/일반버튼

button 은 기본적으로 폼의 정보를 전송하는 역할의 버튼임
type="reset"으로 폼의 정보를 초기화 할 수 있고,
type="button"으로 폼과는 무관한 일반 버튼이 될 수 있다.
버튼의 글자는 시작과 종료 태그 사이에 입력.
